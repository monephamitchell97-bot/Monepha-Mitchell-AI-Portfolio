# Lab 05 – CNN Image Classification: Chihuahua vs Muffin

This project used a Convolutional Neural Network (CNN) to classify images as either a chihuahua or a muffin.

## Topics Covered
- CNN architecture
- Image preprocessing
- Data augmentation
- Deep learning for image classification

## Skills Demonstrated
- Computer vision
- CNN implementation
- Debugging and preprocessing
- Model training and evaluation
